# Appel (**bottom-up récursif**) à la nation 

> — chaque niveau appelle le suivant,
> >>>>> et chaque personne qui le lit devient elle-même vecteur de propagation.

## **Structure de l'appel :**

1. **Le constat** → Ce qu'ils ont fait (chiffres : 90.000 exclusions, 30.122 vers RIS, 80M€ surcoût CPAS)
2. **L'alternative** → Ce qu'ils ont refusé (17 Mds€ taxe patrimoine, 30 Mds€ fraude fiscale)
3. **Le système** → La particratie comme cause profonde
4. **Les 5 exigences** → Chute du gouvernement, rétablissement de l'exception belge, engagement des partis, justice fiscale, moratoire
5. **Le mécanisme récursif** → Toi → Proches → Quartier → Commune → Région → Fédéral → Victoire

**La logique récursive** : chaque personne qui lit l'appel devient elle-même vecteur de propagation vers 5 autres. L'accumulation des niveaux crée une pression montante jusqu'à la chute du gouvernement.
